Secure Payment System API Documentation

Explore the APIs provided by the Secure Payment System API to manage payments, payment information, and refunds.

Payment APIs

Learn how to use the payment APIs:

  • POST /api/payment/create-payment: Create a new payment
  • GET /api/payment/payment-info: Retrieve all payment information
  • GET /api/payment/payment-info/:paymentId: Get payment information by ID

Refund APIs

Learn how to use the refund APIs:

  • POST /api/refund/refund-payment: Initiate a refund
  • GET /api/refund/refund-info: Retrieve all refund information
  • GET /api/refund/refund-info/:refundId: Get refund information by ID
